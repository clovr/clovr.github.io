Start CloVR

  1. DIAG cloud website: http://diagcomputing.org/
  2. Login using course credentials
  3. Select: My Account: Start CloVR
  4. Start CloVR with: Launch CloVR
  5. Check status: My Account: Monitor VM’s
  6. Click on VM to access CloVR interface

 

Run CloVR-Microbe

1. Upload data

  1. On Data sets tab: Add!
  2. Paste URL: http://cb2.igs.umaryland.edu/microbe.tgz
  3. File Type: “Nucleotide FASTQ”
  4. Name: e.g. “illumina_test”

2. Configure Pipeline

  1. Select from applications panel: Microbe
  2. Select Sequencing Dataset(s): “illumina_test”
  3. Select a CloVR Microbe Track: “Assembly+Annotation”
  4. Read Length: “Short”, Read Type: “Paired End”
  5. Account: “local”
  6. Organism: e.g. “Acinetobacter baylyi”
  7. Pipeline Description: e.g. “illumina_test”
  8. Submit

 

Run CloVR-Comparative

1. Upload data

  1. On Data sets tab: Add
  2. Paste URL: http://cb2.igs.umaryland.edu/compare.tgz
  3. File Type: “GenBank”
  4. Name: e.g. “neisseria_5genomes_test”

2. Configure Pipeline

  1. Select from applications panel: Comparative
  2. Input Genbank Tags: “neisseria_5genomes_test”
  3. Account: “local”
  4. Pipeline Description: e.g. “neisseria_test”
  5. Submit
